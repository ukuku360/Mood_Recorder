# 🎹 Mood Recorder

사용자의 기분을 분석하여 실시간으로 음악을 생성하는 인터랙티브 웹 애플리케이션입니다.

## ✨ 주요 기능

1. **기분 입력**: 간단한 문장으로 현재 기분을 표현
2. **AI 무드 분석**: LLM이 사용자의 감정을 분석하고 음악 파라미터로 변환
3. **실시간 음향 조절**: 분석된 무드에 따라 음색, 템포, 필터 등이 자동 조절
4. **키보드 연주**: 아무 키나 눌러서 리듬을 음악으로 실시간 변환

## 🚀 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 환경 변수 설정 (선택사항)

더 정확한 무드 분석을 위해 OpenAI API 키를 설정할 수 있습니다:

```bash
# .env 파일 생성
cp .env.example .env

# .env 파일을 열어서 API 키 입력
VITE_OPENAI_API_KEY=your_api_key_here
```

> **참고**: API 키가 없어도 기본 키워드 기반 무드 분석으로 작동합니다.

### 3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 표시되는 주소(보통 `http://localhost:5173`)로 접속하세요.

## 📖 사용 방법

1. **기분 입력**: 텍스트 상자에 현재 기분을 입력하고 "기분 분석하기" 버튼 클릭
2. **결과 확인**: 분석된 무드와 파라미터(강도, 밝기, 속도)를 시각적으로 확인
3. **연주하기**: 키보드의 아무 키나 눌러서 음악 생성
   - 숫자 키 (1-8): 첫 번째 옥타브
   - QWERTY 행: 두 번째 옥타브
   - ASDF 행: 세 번째 옥타브
   - ZXCV 행: 네 번째 옥타브

## 🎨 무드 카테고리

- **행복 (Happy)**: 밝고 경쾌한 사인파, C 메이저 스케일
- **슬픔 (Sad)**: 느리고 차분한 사인파, C 마이너 스케일
- **평온 (Calm)**: 안정적인 삼각파, 펜타토닉 스케일
- **활기찬 (Energetic)**: 빠르고 활기찬 톱니파, 리디안 스케일
- **화남 (Angry)**: 강렬한 사각파, 디미니시드 스케일

## 🛠️ 기술 스택

- **프론트엔드**: React + TypeScript + Vite
- **오디오 엔진**: Tone.js (Web Audio API)
- **LLM**: OpenAI GPT-3.5 (또는 키워드 기반 폴백)
- **스타일링**: CSS

## 📁 프로젝트 구조

```
src/
├── components/          # React 컴포넌트
│   ├── MoodInput.tsx   # 기분 입력 UI
│   ├── AudioVisualizer.tsx  # 무드 시각화
│   └── KeyboardPlayer.tsx   # 키보드 연주 인터페이스
├── services/           # 핵심 로직
│   ├── llmService.ts   # LLM API 통신
│   ├── audioEngine.ts  # Tone.js 오디오 엔진
│   └── moodMapper.ts   # 무드 → 오디오 파라미터 매핑
├── types/              # TypeScript 타입 정의
└── App.tsx             # 메인 앱 컴포넌트
```
